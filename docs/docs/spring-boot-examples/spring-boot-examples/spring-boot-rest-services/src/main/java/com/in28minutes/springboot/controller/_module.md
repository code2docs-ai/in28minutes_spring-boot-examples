# 说明

StudentController负责处理与学生课程相关的各类请求，主要功能包括获取学生课程列表、处理学生注册课程的请求以及查询特定课程的详细信息。该控制器确保学生能够顺利访问和管理其课程信息，提供全面的课程操作支持。
