# 说明

StudentService类负责管理学生及其相关课程信息，主要提供查询和添加学生及课程的功能。此外，该类还具备处理异常的能力，确保在操作过程中能够有效应对可能出现的错误或异常情况，从而保证系统的稳定性和数据的完整性。
